# Discord Bot 🤖

Basit Discord bot örneği.

## Komutlar

- `a!ping` - Bot latency'sini göster
- `a!hello` - Selamla
- `a!help` - Komutları göster
- `a!eval <kod>` - Kod çalıştır (Owner only)

## Kurulum

```bash
pip install -r requirements.txt
python bot.py
```

## Gereklilikler

- Python 3.8+
- discord.py 2.3.2+
